## 基本类型

1. Number 数值
2. String 字符串
3. Boolean 布尔值
4. Undefined 未定义
5. Null 空
6. Symbol 原始值 唯一 不可变

## 引用类型

1. Object 对象
2. Array 数组
3. Function 函数

### 基本类型和引用类型的区别

存储位置不同

1. 基本类型的值存放在栈中 在栈中存放的是对应的值

2. 引用类型的对象存储于堆中 在栈中存放的是指向堆内存的引用

!!! success TIPS
简单类型赋值，是生成相同的值，两个对象对应不同的地址
复杂类型赋值，是将保存对象的内存地址赋值给另一个变量。也就是两个变量指向堆内存中同一个对象
!!!

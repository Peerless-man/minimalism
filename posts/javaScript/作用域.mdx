## 什么是作用域

简单来说 作用域是在一定区域内可访问变量的集合

## 划分作用域

### 全局作用域

全局作用域的变量在程序的任何地方都能访问

```
let name = '123'

console.log(name) // 123 name没有被声明在函数、块代码里 所以在任何地方都可以访问到
```

### 局部作用域

#### 函数作用域

函数作用域的变量只能在当前函数里被访问 函数外部无法访问

#### 块级作用域

块级作用域的变量只能在当前块里被访问 比如在 {} 里使用 let、const 声明的变量只能在当前 {} 里使用 var 申明的变量没有块级作用域 它会提升到最近的作用域里去

```js
const func = () => {
	if (true) {
		var name = '234'
		let age = 20
	}
	// 由于 var 声明的 name 不存在块级作用域 被提升到了函数作用域 所以能访问 name
	// let 声明的 age 存在块级作用域 不能被提升到函数作用域 所以 {} 外没有 age 的定义
	console.log(name) // 234
	console.log(age) // Uncaught ReferenceError: age is not defined
}
```

!!! note TIPS
面试回答的重点是什么是作用域 以及有哪些作用域(全局作用域、函数作用域、块级作用域) 解释各自的范围
!!!

## 作用域链(拓展)

当在`javascript`中使用一个变量时 首先`javascript`引擎会尝试在当前作用域下去寻找 如果没有找到 就会到上层作用域查找 直到找到全局作用域
